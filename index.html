Application
Some informations about this application

1. The Application

This application follows the practices learned during the Geoweb Application Development discipline. Here were used technologies involving GeoServer, PostGIS, QGIS, OpenLayers and web languages (HTML, CSS, JS).

2. The content

The content used in the application comes from IGBE servers except the layer "Airports". This layer contains the main national airports and it was created by using the QGIS software and then exported to PostGIS. Next we defined the workspace, the store and the layer. So the plotted data have become part of a service that provides the visualization of data to users.

2.1. Layer Airports

The layers named "Airports" was created manually in QGIS and it represents the main airports in Brazil.

2.2. Layer Capitals by population

The layers named "C02_capitais_por_populacao_2010" comes from IGBE services data and it provides the population of each capital in Brazil in the year of 2010.

2.3. Layer Stranger Imigrants

The layers named "C05_imigrantes_estrangeiro_2010" comes from IGBE services data and it provides the number of immigrant strangers in Brazil in the year of 2010.

Web services...

Web service is a solution used in systems integration and communication between different applications. With this technology it is possible that new applications can interact with those that already exist and that systems developed on different platforms are compatible. Web services are components that allow applications to send and receive data in XML format.
